Rails 3.1 dictionary!

Oct 07, 2011

Just released a new version of Rails 3.1 documentation for Dictionary.app together with rubydictionary gem. This gem is a RDoc formatter that turns any ruby documentation into a dictionary.

Download your copy of Rails 3.1 documentation here.

Installation instructions

Unzip downloaded file into ~/Library/Dictionaries folder if you want to keep this dictionary only for yourself. You may need to create this folder if you haven't installed any dictionaries before.
